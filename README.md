## HexToBase64
- clone this repository
    ```bash
    git clone https://github.com/marcoguiidi/hexToBase64.git
    cd hexToBase64
    ```
- compilation
    ```bash
    javac -cp ./libs/commons-codec-1.17.1.jar HexToBase64.java
    ```
- execution
    ```bash
    java -cp ./libs/commons-codec-1.17.1.jar HexToBase64.java
    ````
